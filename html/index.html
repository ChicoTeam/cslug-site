Welcome to CSLUG

The Chico State Linux User Group is an official California State University, Chico club which promotes and assists with the use of the GNU/Linux operating system. CSLUG consists of Linux masters and newcomers alike, and joining is a great first step into the world of Linux.

CSLUG meets twice monthly in O'Connell 211.
